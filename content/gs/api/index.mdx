# How to use the API
import { Tabs } from 'nextra/components'

## Introduction
GroupSecurity API can be used to create custom security engines, or
to integrate GroupSecurity with other plugins. This page will guide you
through the process of using the API.

## Adding the API to your project
<Tabs items={['maven', 'gradle']}>
    <Tabs.Tab>
        ```xml
        <repositories>
            <repository>
                <id>mikart</id>
                <url></url>
            </repository>
        </repositories>
        <dependencies>
            <dependency>
                <groupId>eu.mikart.groupsecurity</groupId>
                <artifactId>GroupSecurity-PLATFORM</artifactId>
                <version>VERSION</version>
            </dependency>
        </dependencies>
        ```
    </Tabs.Tab>
    <Tabs.Tab>
        ```gradle
        allprojects {
            repositories {
                ...
                maven { url '' }
            }
        }
        dependencies {
            implementation 'eu.mikart.groupsecurity:GroupSecurity-PLATFORM:VERSION'
        }
        ```
    </Tabs.Tab>
</Tabs>

## Spigot & Paper
<Tabs items={['paper', 'spigot']}>
    <Tabs.Tab>
        Make GroupSecurity load before your plugin inside the `paper-plugin.yml` like this:
        ```yaml
        dependencies:
            server:
                GroupSecurity:
                    load: BEFORE
        ```
    </Tabs.Tab>
    <Tabs.Tab>
        Make GroupSecurity load before your plugin inside the `plugin.yml` like this:

        ```yaml
        softdepend: [GroupSecurity]
        ```
    </Tabs.Tab>
</Tabs>

## Velocity
Mark GroupSecurity as a dependency in your main Velocity class like this:
```java
@Plugin(id = "your-plugin", name = "Your Plugin", version = "1.0", dependencies = {
    @Dependency(id = "groupsecurity")
})
```
