# Creating a town

<Callout>This page is still under construction. Please check back later.</Callout>

## 1. How to create a town

To create a town, type `/town create (name)`. This command will create a town with the name you specified.

## 2. Claiming land

To claim land for your town, type `/town claim`. This will claim the chunk you are standing in for your town.

## 3. Customizing your town

To customize your town, type `/town`, and browse around the menus to customize what you want.

## 4. Managing your town

If you want to add more players to your town, type `/town trust (player)`. This will add the player to your town.
