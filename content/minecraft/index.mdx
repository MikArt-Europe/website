# Introduction
MikArt Europe Minecraft Network is a place that's made to be a fun and awesome environment for everyone. You can join
our server by using the ip `play.mikart.eu` and selecting the version `1.21.1` or newer.
If you are unable to connect you should ask for help on our Discord.

## What rules are in place?

You can find the rules [here](/help-center/minecraft/rules).

### What mods can I use?

You can use any mods that don't give you an unfair advantage over other players.
If you're unsure if a mod is allowed, check [allowed-modifications](/help-center/minecraft/allowed-modifications)
